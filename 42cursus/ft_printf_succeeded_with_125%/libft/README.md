# libft

## Description
Made a small C library called **libft**. It has basic C functions like string, memory, and linked list stuff. Just learning how things work in C.

## Instructions
Compile with:  
```bash
make
make clean
make fclean
make re
```
## Library 

### String and memory functions
ft_strlen, ft_memcpy, etc.

### Linked list functions 
ft_lstnew, ft_lstadd_front, ft_lstsize, etc.
